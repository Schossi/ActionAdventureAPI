AdventureExtras.SoulsHeroApe Class Reference

Detailed Description

character script for the souls variant of the ape boss fight
performs a random sequence of actions(punch, slam, rage) which costs it energy
when the energy runs out it performs a recovery action that refills it

https://adventure.softleitner.com/manual/character

AdventureCore.TriggerDamageSender Class Reference

Detailed Description

damage sender that damages TriggerDamageReceiver when unity trigger events occur (OnTriggerEnter/OnTriggerExit(2D))

https://adventure.softleitner.com/manual/damage

Member Enumeration Documentation

◆ DirectionMode

enum AdventureCore.TriggerDamageSender.DirectionMode
Enumerator
Forward 

the blue forward axis of the transform the sender is located on

Up 

the green up axis of the transform the sender is located on

Right 

the red right axis of the transform the sender is located on

SenderToReceiver 

direction is calculated from sender to receiver, if a character is associated the character position is used

SenderToReceiverXZ 

direction is calculated from sender to receiver, if a character is associated the character position is used, removes z component so characters are not removed from ground

SenderTransformToReceiver 

direction is calculated from sender to receiver, if a character is associated with the receiver the character position is used

SenderTransformToReceiverXZ 

direction is calculated from sender to receiver, if a character is associated with the receiver the character position is used, removes z component so characters are not removed from ground

None 

no damage vector, target will not be pushed
